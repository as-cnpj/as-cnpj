# Checklist de Release

Langues : [Português (Brasil)](release-checklist.md) | [English](release-checklist.en.md) | [Español](release-checklist.es.md) | **Français**

- relire le contenu public ;
- supprimer tout langage interne de seed ou d’extraction ;
- vérifier métadonnées, liens et licence ;
- exécuter les vérifications syntaxiques ;
- exécuter toute la suite de tests ;
- ne publier que lorsque le contenu et le paquet sont prêts.
