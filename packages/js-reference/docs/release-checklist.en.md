# Release Checklist

Languages: [Português (Brasil)](release-checklist.md) | **English** | [Español](release-checklist.es.md) | [Français](release-checklist.fr.md)

- review public-facing content;
- remove internal seed/extraction wording;
- verify metadata, links and license;
- run syntax checks;
- run the full test suite;
- publish only when content and package are ready.
