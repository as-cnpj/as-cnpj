# Checklist de Release

Idiomas: [Português (Brasil)](release-checklist.md) | [English](release-checklist.en.md) | **Español** | [Français](release-checklist.fr.md)

- revisar el contenido público;
- eliminar lenguaje interno de seed o extracción;
- verificar metadatos, enlaces y licencia;
- ejecutar chequeos sintácticos;
- ejecutar la suite completa de pruebas;
- publicar solo cuando el contenido y el paquete estén listos.
