# Politique de Sécurité

Langues : [Português (Brasil)](SECURITY.md) | [English](SECURITY.en.md) | [Español](SECURITY.es.md) | **Français**

## Portée

Cette politique couvre :

- le noyau de validation de l’écosystème AS-CNPJ ;
- les vecteurs de test partagés ;
- les scripts de build et d’extraction du hub.

## Signalement d’une vulnérabilité

N’ouvrez pas d’issue publique.

Canal préféré :

- GitHub Security Advisories dans le dépôt concerné.

Canal alternatif :

- `ascnpj@0moura.io`

## Objectifs de réponse

- accusé de réception sous 3 jours ouvrés ;
- évaluation initiale sous 7 jours ouvrés ;
- mitigation ou correctif sous 30 jours après confirmation, lorsque applicable.
