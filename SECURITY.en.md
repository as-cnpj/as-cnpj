# Security Policy

Languages: [Português (Brasil)](SECURITY.md) | **English** | [Español](SECURITY.es.md) | [Français](SECURITY.fr.md)

## Scope

This policy covers:

- the validation core of the AS-CNPJ ecosystem;
- shared test vectors;
- build and extraction scripts in the hub.

## Reporting a vulnerability

Do not open a public issue.

Preferred channel:

- GitHub Security Advisories in the affected repository.

Alternative channel:

- `ascnpj@0moura.io`

## Response targets

- acknowledgement within 3 business days;
- initial assessment within 7 business days;
- mitigation or fix within 30 days after confirmation, when applicable.
